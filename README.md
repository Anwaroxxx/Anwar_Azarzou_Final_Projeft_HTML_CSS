## ğŸ’» Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/Anwaroxxx/Anwar_Azarzou_Final_Projeft_HTML_CSS.git
cd Anwar_Azarzou_Final_Projeft_HTML_CSS
```
2. Install Node.js

Download Node.js from [nodejs.org](https://nodejs.org/).

Check installation:

```bash
node -v
npm -v
```

3. Install Sass
```bash
npm install -g sass
sass --version
```

4. Compile SCSS to CSS
```bash
sass scss/main.scss public/css/app.css --watch
```
